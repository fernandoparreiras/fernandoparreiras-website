# Fernando Parreiras Website

Personal website for Fernando Parreiras, built as a public-facing hub for leadership, AI products, systems thinking, mentoring, and Tech Human initiatives.

## Positioning

Fernando Parreiras is an entrepreneur, AI builder, and systems architect focused on practical technology, scalable digital platforms, and human-centered business transformation.

This repository supports the personal website:
